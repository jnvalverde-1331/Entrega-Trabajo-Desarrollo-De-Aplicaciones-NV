Bordes

Los bordes en HTML son líneas que rodean elementos configuradas principalmente mediante CSS para definir su estilo, grosor y color. Permiten estructurar visualmente la página, separando contenidos y destacando áreas con propiedades como border-style, border-width y border-color.

Sombras

Se utilizan para agregar efectos de sombra a textos y cajas usando box-shadow y text-shadow.

Cajas

Permiten trabajar con el modelo de caja en CSS, incluyendo márgenes, padding y tamaños.

Posición

Controla la ubicación de los elementos con propiedades como relative, absolute y fixed.

Transformación

Permite rotar, escalar o mover elementos usando transform en CSS.

Formularios

Se aplican estilos a inputs, botones y formularios para mejorar su apariencia.

Iframe

Permite incrustar otras páginas web dentro de un documento HTML.

Transiciones

Permiten animar cambios de propiedades CSS de forma suave.

Columnas

Permiten dividir el contenido en varias columnas usando CSS3.

Video

Permite insertar y controlar videos dentro de la página web.

Audio

Permite reproducir sonidos y música en una página web.

Transparencias

Permiten controlar la opacidad de elementos usando opacity o rgba.

Animación

Permite crear movimientos y efectos dinámicos usando @keyframes.
