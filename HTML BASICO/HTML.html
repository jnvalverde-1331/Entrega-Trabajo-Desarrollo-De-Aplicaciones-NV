APLICACIONES WEB

Permiten a los usuarios acceder a servicios desde cualquier dispositivo con conexión a internet.

BROWSER (NAVEGADOR)

Un browser o navegador web es un software que permite acceder, interpretar y visualizar páginas web. Su función principal es traducir el código HTML, CSS y JavaScript en contenido visual.

HTML

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Se utiliza para estructurar el contenido mediante etiquetas como títulos, párrafos, imágenes y enlaces.

CSS

CSS (Cascading Style Sheets) se utiliza para dar estilo a las páginas web. Permite cambiar colores, tamaños, posiciones y hacer que el diseño sea más atractivo y organizado.

JAVASCRIPT

JavaScript es un lenguaje de programación que permite agregar interactividad a las páginas web, como botones dinámicos, formularios interactivos y animaciones.

URL

Una URL (Uniform Resource Locator) es la dirección única que se utiliza para acceder a un recurso en internet. Ejemplo: https://www.google.com

CLIENTE / SERVIDOR

Es un modelo de comunicación donde el cliente (navegador del usuario) solicita información y el servidor responde enviando los datos necesarios para mostrar la página web.

HOSTING

El hosting es el servicio que permite almacenar una página web en un servidor para que esté disponible en internet.

DOMINIO

El dominio es el nombre único que identifica un sitio web en internet, como por ejemplo: google.com.

RESPONSIVE DESIGN

El diseño responsivo permite que una página web se adapte a diferentes dispositivos como celulares, tablets y computadores.
