¿Qué es iframe?

La etiqueta iframe permite insertar contenido externo dentro de una página web, como otras páginas, videos o mapas. Es muy utilizada en servicios como YouTube y Google Maps.

Sintaxis básica

Se utiliza el atributo src para indicar la URL del contenido, además de width y height para el tamaño.
